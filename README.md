## About

This repository contains the complete documentation for Esor, a modern web framework based on native Web Components with signals-based reactivity.

## Documentation Structure

- **Getting Started** - Introduction, quick start, and interactive playground
- **Core Concepts** - Tutorial and components guide
- **API Reference** - Complete API documentation and reactivity system
- **Resources** - Templates, starter kits, and practical examples

## Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run docs:dev

# Build for production
npm run docs:build

# Preview production build
npm run docs:preview
```

The dev server will be available at `http://localhost:5173`

## Deployment
