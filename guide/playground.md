## Features

- **Live Editor** - Edit HTML and JavaScript in real-time
- **Instant Preview** - See your changes immediately
- **Multiple Examples** - Switch between different example projects
- **Syntax Highlighting** - Beautiful code editor with syntax highlighting
- **Resizable Panels** - Adjust editor and preview sizes to your preference

## Tips

- Click on any file in the file tree to edit it
- The preview updates automatically as you type
- Try modifying the examples to experiment with Esor's features
- Check the browser console for any errors or logs
